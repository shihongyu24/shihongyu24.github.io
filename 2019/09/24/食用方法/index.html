1、安装git、node.js和hexo

git

分布式版本管理系统，也就是管理文件不同版本的系统。
官方网站：https://gitforwindows.org/

node.js

在服务器上跨平台的JavaScript执行环境。
官方网站：https://nodejs.org/en/

node.js -v #查看版本

在安装完node.js后在cmd里输入指令把npm（node包的管理器）替换成淘宝镜像，安装插件会变快。

创建、设置SSH

登录github网站：https://github.com/

在GitHub网站里进入setting-SSH and GPG keys添加新的ssh key。title随便取，把刚才复制的部分放进key里。
